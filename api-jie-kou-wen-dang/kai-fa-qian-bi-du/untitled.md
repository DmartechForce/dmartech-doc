description: 开放平台每次调用接口时，可能获得正确或错误的返回码，开发者可以根据返回码信息调试接口，排查错误。
---

# 全局返回码说明

| 返回码 | 说明 |
| :--- | :--- |
| -1 | 系统繁忙，此时请开发者稍候再试 |
| 0 | 请求成功 |
| 10001 | 本次操作出现异常，请稍后再试。 |
| 10002 | 访问未授权或者授权信息已失效 |
| 10003 | 无效排序字段 |
| 10004 | 参数值无效 |
| 10005 | 无效查询字段 |
| 10006 | 无效排序类型 |
| 20001 | 缺少必要参数 |
| 40001 | accessKey无效为空/accessSign无效/签名错误 |
| 40003 | 没有接口权限访问 |
| 40006 | 接口调用次数/频率达到限制 |
| 11001 | 不合法的长度限制 |
